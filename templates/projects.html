1. Full-Stack Database Projects

BUG TRACKER

This is a ticket tracking system that could be used by e.g. a sofware company to keep track of employees, projects and tickets. The program features multi-role authentication and authorization, demo-login and an automated notification system. I've build this project from scratch, including relational database design and a responsive front-end.

The back-end is build with (object-oriented) PHP, raw SQL and a MySql-database. On the front-end, I've build everything up with vanilla HTML, CSS and JS, except for the styling of a few components (tables and pop-ups), where I've used a light-weight CSS-library.

If you only visit once, I recommend that you log in as 'Demo Admin'. If something looks strange in your browser, try Chrome.
